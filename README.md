# 🚀 CRM Sales Dashboard

A modern CRM (Customer Relationship Management) dashboard built using HTML, CSS, JavaScript, Chart.js, and frontend UI concepts.

## ✨ Features

* 🔐 Login & Register Authentication
* 📊 Sales Analytics Dashboard
* 📋 Deal Management System
* 🧩 Kanban Workflow
* 📅 Gantt Timeline
* 🤖 AI Insights UI
* 📈 Charts & Reports
* 🎨 Responsive Modern UI

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
* Chart.js
* NestJS Backend

## 📂 Modules

* Dashboard
* Records
* Kanban
* Actions
* App Description
* AI Insights
